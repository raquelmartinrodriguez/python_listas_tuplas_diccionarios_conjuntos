# Actividad 31 - Tipos de Datos Avanzados en Python

## Descripción

Esta actividad forma parte del **Máster en Bioestadística y Bioinformática** y tiene como objetivo practicar el uso de las principales estructuras de datos de Python:

* Listas
* Tuplas
* Conjuntos (Sets)
* Diccionarios

Además, se incluyen ejercicios relacionados con la manipulación de datos biomédicos simulados, tales como pacientes, síntomas, medicamentos y tratamientos.

---

## Contenido de la actividad

### 1. Listas y Tuplas

Se han realizado operaciones básicas sobre listas y tuplas:

* Creación de estructuras de datos.
* Acceso a elementos mediante índices.
* Modificación de elementos (cuando es posible).
* Búsqueda de elementos.
* Obtención del tamaño.
* Adición y eliminación de elementos.

Ejemplos utilizados:

* Medicamentos.
* Síntomas de pacientes.

---

### 2. Sets y Diccionarios

Se han utilizado conjuntos y diccionarios para representar información clínica:

* Creación de conjuntos de medicamentos.
* Creación de diccionarios con información de pacientes.
* Búsqueda de elementos.
* Modificación y ampliación de datos.
* Eliminación de contenido.

Ejemplos utilizados:

* Datos de pacientes.
* Tratamientos farmacológicos.

---

### 3. Cálculo de Temperatura Media

Programa que:

1. Solicita tres temperaturas al usuario.
2. Almacena los valores en una lista.
3. Calcula el sumatorio mediante la función `sum()`.
4. Obtiene el promedio utilizando `len()`.

Conceptos trabajados:

* Entrada de datos.
* Conversión a tipo `float`.
* Operaciones matemáticas.
* Listas.

---

### 4. Operaciones con Conjuntos

Se han realizado operaciones de teoría de conjuntos utilizando pacientes pertenecientes a diferentes estudios clínicos.

Operaciones realizadas:

* Eliminación de elementos.
* Adición de nuevos pacientes.
* Intersección entre conjuntos.

Ejemplo:

* Identificación de pacientes presentes en ambos estudios.

---

### 5. Gestión de Tratamientos Médicos

Se ha trabajado con un diccionario que almacena:

* Identificador del medicamento.
* Nombre del tratamiento.
* Cantidad administrada.

Tareas realizadas:

* Visualización del tratamiento.
* Incremento de dosis en un 50%.
* Verificación de los cambios realizados.

---

## Tecnologías utilizadas

* Python 3
* Jupyter Notebook (.ipynb)
* Visual Studio Code
* Git y GitHub

---

## Objetivos de aprendizaje

* Comprender las diferencias entre listas, tuplas, conjuntos y diccionarios.
* Manipular estructuras de datos en Python.
* Aplicar operaciones básicas sobre colecciones.
* Utilizar estructuras de datos en ejemplos relacionados con el ámbito biosanitario.
* Desarrollar buenas prácticas de programación y documentación.
